# RelayGo - 轻松创建 Telegram 私聊机器人

## 🔥 项目简介

RelayGo 是新一代 Telegram 私聊机器人，支持人机验证、联合封禁、自定义欢迎消息和自动回复等多个功能。

机器人采用 Serverless 架构，无需服务器即可部署，简单而强大！部署教程请查看 [DEPLOY.md](https://github.com/abcxyz-123456/RelayGo/blob/main/DEPLOY.md) 。

## ✨ 功能特性

### 私聊转发

- 自动将用户私聊消息转发到群组话题（Topics）
- 一人一个独立话题，互不干扰
- 管理员可在群内直接回复用户

### 联合封禁

- 接入中心化封禁系统
- 多机器人共享黑名单
- 恶意用户一次封禁，全网拦截

### 人机验证

- 本地验证模式支持：算数验证、贴纸验证。
- 开启联合封禁后使用 Turnstile 人机验证
- 有效防止垃圾消息

### 自定义欢迎消息

- 自定义用户首次收到的消息
- 支持设置欢迎消息按钮

### 设置自动回复

- 用户发言自动回复自定义内容
- 每 10 分钟仅触发一次，避免刷屏

### 发布广播

- 批量向所有用户发送通知
- 自动跳过封禁用户

### 手动封禁/解封

- 管理员手动封禁/解封用户
- 支持群内或私聊操作
- 黑名单在本地，即时生效

### Serverless 架构

- 无需购买服务器
- 无需独立数据库
- 免费额度即可支持中小规模长期运行

## 🧠 工作原理

RelayGo 基于 Telegram Topics 机制实现「私聊 → 群组话题」的消息桥接：

```text
用户私聊
   ↓
群组 Topic（独立会话）
   ↓
管理员回复
   ↓
自动回传用户
```


每个用户对应一个独立话题，实现类似客服系统的体验。

## 🚀 适用场景

适合：

- 匿名投稿机器人
- 私聊中转机器人
- 反馈收集机器人
- 社群接待机器人
